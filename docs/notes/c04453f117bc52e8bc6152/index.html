组二：时间戳 + 六种模式状态（本版全局参数）

纸上应出现的内容

在 S / E / C 下方追加模式块，例如：

S:12345
E:14123
C:1778ms
Modes:
* anime
- cartoon
- ink_wash
- line_art
- simple_drawing
- sketch
  • * 前缀：当前 UI 选中的模式
  • - 前缀：其余五种模式（顺序与 lv_example_meter.c 中 s_mode_names[] 一致）

本版（当前仓库）speed=5 全局参数纪要

对应 applications/Printer/thermal_printer.c 中 apply_speed_profile() 的 default /* 5 fastest */ 分支及位图循环相关逻辑：

项数值
inter_group_us1
line_cooling_us10
heat_us_at_5c / 25c / 40c1080 / 940 / 820
motor_step_us650
status_check_period_lines96
连续白行批量上限96
周期性 rt_thread_mdelay(1)约每 256 行
位图结束后 thermal_printer_feed2
高密度行额外加热（黑点档）+135 / +90 / +55 / +28 / +10（分档）

与组一的差异（便于分析）

  • 纸上多出 Modes 文本块（由 thermal_printer_print_text → 内部再走位图，会略增加总时长与走纸，属正常现象）。
  • 热参数与白行批量相对组一更激进，同图下 C 通常应 ≤ 组一；若画质下降，可对照两组样张与参数表回调。

  • 文件名建议：YYYYMMDD_纸上Modes行首模式名_含Modes_备注.jpg（与 * 行一致即可）。
