2. 业务主链路（与文档的对应关系）

便于从「功能」反查「写了哪篇记录」：

  1. 启动与网络：main.c → 架构评估文档 §2、§P0-2。
  2. MQTT 控制面 / 二进制收图 / 打印：mqtt_backend_* → 05-01 / 05-03 记录；架构评估 §P1-2。
  3. 热敏打印与时序标定：thermal_printer.c + prn_tune 文档 + 参数排列组合记录 + 05-07 算法分析。
  4. 按键：P37 语音长按、P36 二维码 —— button_control.c；语音与 MP3 纪要；BUG-001。
  5. LVGL 桌面与二级页：lv_example_meter.c、ui_strings、thermal_happy_lv —— 05-06 / 05-07 三篇 LVGL 记录。
  6. 多语言：device_config + ui_strings + 字体脚本 + BK7252n 上 LVGL 多语言界面开发指南。
  7. OTA：OTA_Upgrade/、fal_cfg —— 桌面二维码/WS 总结中有索引性说明。
  8. 按键蜂鸣 / 生图等待旋律 / 音量与内存：beep_* → 05-18 初版记录；05-27 增量（DMA、TCM、volumeSet、收图打印衔接）见 蜂鸣/旋律 MQTT 协同与 TCM 内存优化开发记录。

4. 已知风险与后续事项（摘自现有记录）

  • BUG-001：MQTT 二进制共享缓冲与打印 / LVGL 异步预览的并发覆盖风险；关闭标准见 bug-collector/BUG-001-*.md。
  • 架构：MQTT 凭证硬编码、启动无限等待、打印为高占用 bit-bang 等 —— 见架构评估文档分级建议。
  • 打印算法：delay_us 与 CPU 频率耦合、dense_boost 与 gshape 叠加等 —— 见 05-07 算法分析中的改进优先级。
  • ASR：帧头 8 字节、采样率字段、QoS0 丢包等 —— 见语音/MP3 排查总结 §4。
